# Contributor Covenant Code of Conduct

## Наше обязательство

Мы стремимся сделать участие в проекте свободным от притеснений для всех.

## Стандарты

Примеры позитивного поведения: уважительный тон, конструктивная критика, признание чужого вклада.

Неприемлемо: троллинг, оскорбления, публикация личной информации без согласия, иное неэтичное поведение.

## Применение

О нарушениях можно сообщить владельцу репозитория (см. [AUTHORS.md](AUTHORS.md)). Жалобы рассматриваются конфиденциально.
