# Why this repository is open

GPT-hub started as a **hackathon learning project** (MWS GPT case, True Tech Arena 2026): assemble a working chat stack from OSS — Open WebUI, routing, optional RAG — and iterate in the open.

It is **not** a claim of a unique product idea; similar integrations exist. The value is a **documented path**: Docker, tests, trace, deployment notes, and an evolution v1→v4 that others can fork.

The hackathon also reinforced that delivery is often about **team coordination** — roles, deadlines, and clear docs — as much as about code depth.

If this snapshot saves you time or helps your team, **fork and use it**. MIT license for our code; third-party OSS under their licenses. Official competition rules always override this repo for submissions.
